# Smartypants plugin

A field method for [Kirby CMS](http://getkirby.com) to apply smartypants (until it’s added to [Kirby CMS](http://getkirby.com)’s core).

## How to use it?

Call from template:

```php
<?php echo $site->title()->smartypants()->widont(); ?>
```
